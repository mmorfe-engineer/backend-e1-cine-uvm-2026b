📋 Sobre el Proyecto

🎯 Objetivo

Sistema backend completo para la gestión integral de un cine: películas, salas, funciones, tickets y reservaciones con persistencia en MySQL y arquitectura MVC.

🏗️ Arquitectura

Patrón MVC con Separación de Responsabilidades. Controladores como clases JavaScript con métodos static async. Promesas en toda la aplicación.

🗄️ Base de Datos

MySQL 8.0 con 5 tablas relacionadas, Foreign Keys, índices de rendimiento, 3 vistas SQL y transacciones atómicas para operaciones críticas.

🚀 Deploy

Backend en Railway con MySQL cloud. GitHub Pages para esta documentación. 14+ commits en GitHub con historial completo del equipo.

⭐ Funciones Técnicas Destacadas

getLast5()

Retorna las 5 funciones más recientes con JOIN a películas y salas. Usa ORDER BY date DESC, time DESC LIMIT 5.

getByDateRange()

Filtra funciones entre dos fechas con BETWEEN ? AND ?. Query parametrizada que previene SQL injection.

cancel() — Transacción

Cancela reservación y libera ticket de forma atómica con beginTransaction → COMMIT / ROLLBACK.
